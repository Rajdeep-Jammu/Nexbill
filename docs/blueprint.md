# **App Name**: Momentum Inventory

## Core Features:

- Secure User & Shop Onboarding: Allows users to create a shop profile, set up a secure PIN (with optional biometric simulation), and securely manage authentication details.
- AI Product Description Generator: Utilizes AI as a tool to assist in generating creative and informative descriptions for new products, enhancing consistency and saving time.
- Comprehensive Product Management: Enables adding, editing, and deleting products with details like name, price, quantity, and image uploads, alongside powerful search, filtering, and visual stock indicators.
- Streamlined Billing Workflow: Facilitates creating new sales bills by adding products to a dynamic cart, automatically calculating subtotal, tax, and total, and assigning customer information.
- Professional Invoice & Report Generation: Generates high-quality PDF invoices (including shop details, product table, totals) and allows for the creation of sales reports (daily/weekly/monthly) with export options.
- Interactive Dashboard Analytics: Provides a visual summary of key business metrics such as total products, inventory value, low stock alerts, today's sales, and weekly sales graphs for quick insights.
- Dynamic Settings & Personalization: Offers options to manage shop details, reset PINs, and adjust user interface preferences like dark/light mode for a personalized experience.

## Style Guidelines:

- Background color: A deep dark navy (#0B0F19) providing a rich, premium foundation, reminiscent of high-end dashboards.
- Primary accent color: An elegant indigo (#6366F1) for primary interactive elements, embodying sophistication and modernity.
- Secondary accent color: A vibrant green (#22C55E) to highlight positive actions, success states, and confirmations.
- Danger/Error color: A bold red (#EF4444) for alerts and destructive actions, ensuring clear communication.
- Card background color: A slightly lighter dark gray (#111827) with soft glassmorphism effects for content cards, maintaining visual hierarchy.
- Text colors: Predominantly white for main content and muted grays for secondary text, ensuring readability against the dark theme.
- Headlines and prominent text will use 'Poppins', a geometric sans-serif, for its clean, contemporary, and bold appearance. Body text and data tables will utilize 'Inter', a grotesque-style sans-serif, for excellent readability and a modern, objective feel.
- Use a consistent set of minimal, clean line icons that align with the premium, SaaS aesthetic of Stripe or Linear, ensuring clarity and unobtrusiveness.
- Mobile-first design with a floating bottom navigation bar featuring a blur and glow effect. Ample, breathable spacing between elements to avoid clutter. User interface elements, including cards and buttons, will feature large 2xl rounded corners and soft shadows for depth and a premium feel.
- Implement smooth page transitions and micro-animations on interactive elements, such as card hovers and button clicks (including a hover glow effect), powered by Framer Motion to enhance the tactile and engaging user experience. Incorporate skeleton loading states for data fetching.
